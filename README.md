# Rock, Paper, Scissors Game

This is a simple Rock, Paper, Scissors game implemented in JS.
This game will is played entirely in the console.

## Features
-The game is played against the computer.
-You can enter your choice in the prompt window in the browser or in the terminal.
-The computer randomly selects its choice.
-The results and the current score are displayed after each round.
-The game ends when either the player or the computer reaches a score of 5, and the winner is announced.

## Rules
-Rock beats Scissors.
-Scissors beats Paper.
-Paper beats Rock.
